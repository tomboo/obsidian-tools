# obsidian-tools

A collection of standalone Python scripts for working with Obsidian vaults.

## Install Pattern

Each tool is installable on its own from this repo:

```bash
uv tool install 'git+https://github.com/tomboo/obsidian-tools.git#subdirectory=tools/<tool-name>'
```

## Scripts

| Script | Description |
|---|---|
| [print_md](tools/print_md/README.md) | Convert Obsidian markdown files to print-ready PDFs |
